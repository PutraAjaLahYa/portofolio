Untuk saat ini saya memiliki sedikit ilmu tentang pembuatan web, database, apk android, menggunakan bahasa HTML, CSS, JS, JAVA, PHP, PYTHON, Memang masih jauh dari kata hebat dan baru mempelajari dasar nya saja dari pendidikan saya tetapi saya selalu bersemangat untuk belajar ilmu coding untuk memperdalam keterampilan saya.

Saya juga mengikuti beberapa pelatihan untuk memperdalam ilmu coding saya di situs situs online dan juga mendapat kan sertifikat namun saya merasa itu masih belum cukup untuk di sebut sebagai programer
